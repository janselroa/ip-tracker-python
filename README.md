### installation

clone repository

```
git clone https://github.com/janselroa/ip-tracker-python.git
```
#### install pipenv
```
pip install pipenv
```

```
pipenv shell
```
#### run program
```
python main.py
```
or in linux/mac

```
python3 main.py
```
